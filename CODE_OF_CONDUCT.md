# 贡献者契约 Code of Conduct

## 我们的承诺

为了营造开放、友好的环境，作为本项目的维护者与贡献者，我们承诺让参与本项目的每一个人都不受骚扰，无论年龄、身体特征、残疾、民族、性别认同与表达、经验水平、国籍、个人外表、种族、宗教或性取向。

## 我们的标准

正面行为示例：

- 使用友好、包容的语言
- 尊重不同观点与经验
- 优雅接受建设性批评
- 关注社区整体利益
- 对其他社区成员保持同理心

不可接受的行为示例：

- 含性暗示的言语或图像，以及不受欢迎的性关注
- 挑衅、侮辱、贬损评论以及人身或政治攻击
- 公开或私下的骚扰
- 未经明确许可发布他人的私人信息（地址、邮箱等）
- 在专业场合可被合理认定为不当的其他行为

## 我们的责任

项目维护者有责任厘清可接受行为的标准，并在出现不当行为时采取合适且公平的纠正措施，包括但不限于：编辑、删除评论、提交、代码、wiki、issue 等贡献，必要时对违反本契约的贡献者临时或永久封禁。

## 适用范围

本契约适用于项目空间内的所有活动，以及个人代表项目或社区出现在公共空间的场合（如使用项目邮箱、官方社交账号、线下/线上活动）。

## 执行

举报骚扰、虐待或其他不可接受行为，可通过 [SECURITY.md](./SECURITY.md) 中的私密渠道联系维护者。所有举报都会被审阅与调查，回应方式视情况而定。维护者有义务保护举报者的身份。

## 出处

本契约改编自 [Contributor Covenant 2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/)。
